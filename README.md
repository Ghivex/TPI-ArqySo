# TPI-ArqySo
En el presente trabajo se aborda el concepto de virtualización, una tecnología fundamental en el ámbito de la informática actual, que permite ejecutar múltiples sistemas operativos en una única máquina física mediante el uso de software especializado. Esta herramienta resulta especialmente relevante en entornos de desarrollo y administración de sistemas, ya que posibilita la creación de entornos aislados sin necesidad de contar con hardware adicional.

La elección de este tema se basa en su importancia dentro de la formación como técnica en programación, dado que la virtualización permite comprender de manera práctica la interacción entre el hardware, los sistemas operativos y las aplicaciones. Además, facilita la experimentación segura con diferentes configuraciones sin comprometer el sistema principal.

El objetivo de este trabajo es analizar los conceptos teóricos fundamentales de la virtualización y aplicar dichos conocimientos mediante la implementación de una máquina virtual utilizando VirtualBox. En este entorno se instalará un sistema operativo Ubuntu y se configurará un servidor web, permitiendo observar el funcionamiento de un servicio real dentro de un sistema virtualizado.

De esta manera, se busca no solo afianzar los conocimientos teóricos, sino también desarrollar habilidades prácticas relacionadas con la instalación, configuración y utilización de entornos virtuales.
